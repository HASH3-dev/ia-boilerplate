# Release v1.0.0 - 2026-05-22

## Summary

First stable release of the Genemunology monorepo boilerplate. Establishes the full project structure for both the API (NestJS) and Web (Next.js) apps, including architecture conventions, containerized development environment, test infrastructure, and the health use-case refactor that enforces consistent handler delegation across all API modules.

## Migration And Deployment Notes

- The unified dev container replaces the previous `dev-backend` setup. Run `npm run dev:build` before `npm run dev` on first use after pulling this release.
- No database migrations are required for this release.
- No new environment variables are required.
- Install dependencies with `npm install` at the monorepo root before running any workspace scripts.
